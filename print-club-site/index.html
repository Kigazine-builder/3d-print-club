3-D printing requests, club approvals, code-based ordering

A club site that runs like a mini Amazon — without payments.

People sign up, the owner approves or rejects requests in an admin page, approved members receive a unique club order code, and members use that code to order 3-D prints from a database-backed product catalog.

Included system

Firebase Auth, Firestore products, user requests, member codes, orders, admin approval tools, and admin product creation.

Membership approval

A new user creates an account and their profile enters pending status until the club owner approves it.

Code checkout

Approved users see a code like JANE-8KD2QZ. That code is required to place orders.

Admin inventory

The owner creates, edits, hides, and deletes products directly from the admin dashboard.
